# Image Sources

The product photos used in this portfolio demo were prepared from Pexels stock photography and stored locally in the project under:

```txt
images/products/catalog/real/
```

Pexels states that photos and videos on Pexels can be downloaded and used for free, attribution is not required, and modification is allowed. Attribution is still appreciated, so this file documents the source category pages used while improving the Sportivo catalog.

## Source categories used

- Pexels soccer ball photos
- Pexels basketball photos
- Pexels table tennis photos
- Pexels volleyball photos
- Pexels sports shoes photos
- Pexels sports bag photos
- Pexels training cones photos
- Pexels volleyball net photos

## Portfolio usage note

This project is a front-end portfolio demo. Product names, prices, inventory counts, descriptions, and ratings are realistic sample data created for the demo experience. They do not represent official inventory, official pricing, or brand endorsement.
